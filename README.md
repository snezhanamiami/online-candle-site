# Онлайн-свеча

Рабочий сайт на Node.js + Express + Socket.IO.

## Возможности

- форма с выбором намерения и вводом имени;
- анимированное пламя;
- свеча постепенно уменьшается и гаснет через 2 минуты;
- все подключённые пользователи видят общий подсвечник в реальном времени;
- один пользователь не может поставить вторую свечу, пока горит первая;
- идентификатор пользователя и время блокировки сохраняются в cookie;
- при наведении показываются имя, намерение и оставшееся время;
- максимум 40 одновременно горящих свечей;
- свеча текущего пользователя выводится на передний план.

## Запуск

```bash
npm install
npm start
```

Открыть: http://localhost:3000

## Важно для публикации

Сейчас активные свечи хранятся в памяти процесса. Для production и нескольких серверных инстансов лучше заменить хранилище на Redis или PostgreSQL и подключить Redis Adapter для Socket.IO.
